## 按大小列出所有pip包命令

```shell
pip list --format freeze | awk -F = {'print $1'} | xargs pip show | grep -E 'Location:|Name:' | cut -d ' ' -f 2 | paste -d ' ' - - | awk '{gsub("-","_",$1); print $2 "/" tolower($1)}' | xargs du -sh 2> /dev/null | sort -h
```

## 按大小列出所有apt包命令

```shell
dpkg-query -Wf '${Installed-Size}\t${Package}\n' | sort -n
```
